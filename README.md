# Stock Inventory App (Next.js + Tailwind)

Simple stock inventory app with:
- Dashboard
- Stock In module
- Stock Out module
- More menu:
  - Movement history with filter (in, out, opname)
  - Manage products module
  - Stock opname module
- Responsive mobile layout with iOS-style bottom navigation and icons
- Event animations (toast + panel/trigger pulse)

No product price fields are used.

## Run

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Build Check

```bash
npm run build
```
